MeasureListener
===============

.. java:package:: org.uma.jmetal.measure

.. java:type:: public interface MeasureListener<Value>

   A \ :java:ref:`MeasureListener`\  allows to register a given behavior to \ :java:ref:`PushMeasure.register(MeasureListener)`\ . When the \ :java:ref:`PushMeasure`\  generate a \ :java:ref:`Value`\ , it is provided to \ :java:ref:`MeasureListener.measureGenerated(Object)`\  in order to execute the specified behavior.
